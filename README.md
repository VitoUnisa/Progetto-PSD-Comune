# Sistema di gestione delle segnalazioni di un comune 

# Descrizione
Programma creato in C che permette la gestione delle segnalazioni di un comune

# Funzionalità:

- Inserimento segnalazioni

- Ricerca per categoria

- Ricerca per codice

- Aggiornamento stato

- Visualizzazione filtrata

- Gestione priorità

- Eliminazione segnalazioni

- Report finale

# Struttura del progetto:

main.c          -> Gestione menù principale

segnalazioni.c  -> Implementazione delle funzioni

segnalazioni.h  -> Definizione della struttura dati e dei prototipi delle funzioni

test.c          -> Test del programma

Makefile        -> Metodo e specifica di compilazione del programma


# Compilazione
Nel cmd: make

# Avvio
Nel cmd: ./main

# Test
Nel cmd: ./test

# Pulizia
Nel cmd: make clean


# Struttura dati utilizzata
Lista concatenata semplice.
